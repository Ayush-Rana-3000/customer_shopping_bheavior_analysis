# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python**, **PostgreSQL**, **SQL**, and **Power BI**. The project demonstrates the complete analytics workflow from data preprocessing to business intelligence dashboard creation.

---

## 📌 Project Overview

Understanding customer purchasing behavior is essential for improving business performance and customer satisfaction.

This project analyzes customer shopping data to answer important business questions related to:

- Customer demographics
- Revenue generation
- Product performance
- Subscription behavior
- Discount usage
- Customer segmentation
- Purchasing trends

The project follows a complete analytics pipeline including:

- Data Cleaning
- Feature Engineering
- Database Integration
- SQL Analysis
- Interactive Dashboard Development
- Business Insights

---

# 🎯 Objectives

- Clean and preprocess customer shopping data
- Handle missing values efficiently
- Perform feature engineering
- Store processed data in PostgreSQL
- Answer business questions using SQL
- Build an interactive Power BI dashboard
- Generate actionable business insights

# 🧹 Data Preprocessing

The following preprocessing steps were performed:

✅ Missing value analysis

✅ Missing Review Rating values replaced using category-wise median

✅ Standardized column names

✅ Removed redundant columns

✅ Converted categorical values

✅ Feature Engineering

---

# ⚙️ Feature Engineering

Two new features were created:

### Age Group

Customers were categorized into:

- Young Adult
- Adult
- Middle Aged
- Senior

---

### Purchase Frequency Days

Purchase frequency categories were converted into numerical values.

Example:

- Weekly → 7
- Bi-weekly → 14
- Monthly → 30

---

# 🗄 Database Integration

The cleaned dataset was exported to PostgreSQL using SQLAlchemy.

The database was then queried using SQL to answer business-oriented questions.

---

# 🔍 SQL Business Analysis

The following business questions were answered.

### 1️⃣ Revenue generated by Male vs Female customers

Determines which gender contributes more revenue.

---

### 2️⃣ Customers using discounts but spending above average

Identifies high-value customers who remain profitable despite discounts.

---

### 3️⃣ Top 5 highest-rated products

Finds products with the best customer reviews.

---

### 4️⃣ Standard vs Express Shipping comparison

Compares average purchase amounts across shipping methods.

---

### 5️⃣ Subscriber vs Non-subscriber spending

Evaluates the effectiveness of subscription programs.

---

### 6️⃣ Products with highest discount usage

Identifies products frequently sold through promotions.

---

### 7️⃣ Customer Segmentation

Segments customers into:

- New
- Returning
- Loyal

---

### 8️⃣ Top 3 products within each category

Ranks products using SQL Window Functions.

---

### 9️⃣ Repeat Buyers vs Subscription

Determines whether repeat customers subscribe.

---

### 🔟 Revenue by Age Group

Measures revenue contribution of different age segments.
